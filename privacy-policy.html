Política de Privacidade - BrowserMonitor

Última atualização: 19 de junho de 2026

1. Sobre o BrowserMonitor

O BrowserMonitor é uma extensão para navegadores baseada no conceito de "caixa preta", permitindo ao usuário visualizar estatísticas sobre seu tempo de navegação em diferentes sites.

2. Dados coletados

A extensão registra apenas informações necessárias para o seu funcionamento:

  • Domínio dos sites visitados (ex.: youtube.com, github.com)
  • Tempo de permanência em cada site
  • Quantidade de visitas registradas
  • Estatísticas de navegação geradas pela própria extensão

3. Armazenamento dos dados

Todos os dados são armazenados localmente no navegador do usuário por meio da API chrome.storage.local.

Nenhuma informação é enviada para servidores externos.

4. Compartilhamento de dados

O BrowserMonitor:

  • Não vende dados.
  • Não compartilha dados com terceiros.
  • Não envia informações para serviços externos.
  • Não utiliza ferramentas de rastreamento ou publicidade.

5. Informações não coletadas

A extensão não coleta:

  • Senhas
  • Dados bancários
  • Cookies
  • Conteúdo de formulários
  • Mensagens
  • Arquivos pessoais
  • Informações de identificação pessoal

6. Controle do usuário

O usuário pode apagar todos os dados registrados a qualquer momento utilizando a opção "Limpar Dados" disponível na própria extensão.

Também é possível remover completamente a extensão através das configurações do navegador.

7. Alterações nesta política

Esta política poderá ser atualizada futuramente para refletir melhorias ou novas funcionalidades da extensão.

A data da última atualização será sempre informada no início deste documento.
